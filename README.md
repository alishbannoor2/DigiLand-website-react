
# Digiland Market

A modern web app starter built with **React + Vite**, styled using **TailwindCSS**, and linted with **ESLint** for clean code. This template provides a fast development experience with hot module replacement (HMR) and optimized production builds.

---

## 🚀 Features
- ⚡ **Fast development** with Vite and HMR  
- 🎨 **TailwindCSS** for responsive, utility‑first styling  
- ✅ **ESLint** rules to keep code consistent and clean  
- 📦 Lightweight setup, ready to expand into full applications  

---

## 🛠 Tech Stack
- **React** – UI library  
- **Vite** – Build tool & dev server  
- **TailwindCSS** – Styling framework  
- **ESLint** – Code linting  

---

## 📂 Project Setup

### Install dependencies
```bash
npm install
```

### Run in development
```bash
npm run dev
```

### Build for production
```bash
npm run build
```

### Preview production build
```bash
npm run preview
```

### Lint code
```bash
npm run lint
```

## 📦 Deployment
- The build output is generated in the `dist/` folder.  
- Configure your hosting (e.g., Vercel) to serve from `dist`.  
- Ensure image imports use `import` syntax so assets are bundled correctly.

---

## 🌟 Getting Started
Clone the repo, install dependencies, and start building your app. This template is designed to be minimal yet flexible, so you can quickly adapt it for your own projects.
